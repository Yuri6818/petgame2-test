## TROUBLESHOOTING

**Page doesn't load?**
- Refresh the browser (F5)
- Check that npm run dev is still running

**Items not showing?**
- Refresh the page (F5)
- Check dev server output for errors

**Button doesn't work?**
- Some buttons are placeholders (Training, Adopt, etc.)
- Shop, Inventory, Familiars, Activities fully work

**Images not loading?**
- Check that your /img/ folder has all assets
- Should auto-fallback to default image
